How to Play

Players:
• Electric Violet - places horizontal tokens
• Shocking Pink - places vertical tokens

Placing Tokens:
• Click/tap empty cells to place tokens
• Each token occupies 2 cells
• Clicked cell = left part (Violet) or top part (Pink)

Controls:
• Tap - place token
• Swipe - rotate view
• Pinch - zoom in/out
• Release - return to north

Game Rules:
• Green cells = playable
• Gray cells = blocked
• Last player to place token wins!
